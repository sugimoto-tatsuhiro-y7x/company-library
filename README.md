# 社内図書システム

## ディレクトリ構成

### views

画面単位でファイルを作成していく。

例：本一覧画面、ユーザ詳細画面

### components

モーダル等、画面内で呼び出されるコンポーネント単位でファイルを作成していく。

例：本詳細モーダル
### layouts

今回は使わない。
